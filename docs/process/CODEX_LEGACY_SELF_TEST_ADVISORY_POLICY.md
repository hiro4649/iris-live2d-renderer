# Legacy Self-Test Advisory Policy

Only the active harness version self-test is blocking for the current release. For v0.9.9, the active self-test status key is `v099SelfTestStatus`.

Legacy self-tests remain useful advisory evidence. They must not block target quality for non-active versions unless the active policy explicitly requires them.

Active v0.9.9 self-test failure is non-overridable.
