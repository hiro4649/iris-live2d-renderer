# Golden Set Policy

Golden cases are deterministic regression fixtures for harness audit behavior.
They use no network, no LLM judge, no external API, and no raw unsafe values in
reports. They cover safe output scanning, evidence pack validation, structured
human confirmation, CI replay classification, generic core mode, AGENTS context,
and Best of N evidence requirements.
