# Codex PR Template Compiler Policy

The PR template compiler emits safe skeletons for known task profiles. It is
suggestion-only and never edits PR bodies, commits, pushes, or posts on its own.

Missing required sections remain gate-relevant. The compiler provides safe
headings and section names so humans can repair the PR body without raw logs or
payloads.
