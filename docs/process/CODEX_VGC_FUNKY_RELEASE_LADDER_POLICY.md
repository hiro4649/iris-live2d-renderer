# VGC FUNKY Release Ladder Policy

VGC-FUNKY release ladder is ordered from contract source evidence through test readiness, dry-run, no-fund simulation, testnet deploy, verification, governance configuration, mainnet preflight, and mainnet go/no-go. No repo creation or deployment occurs in v1.0.3.
