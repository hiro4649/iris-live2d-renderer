# Small Product PR Fast Path Policy

Small product PR fast path reduces duplicate prose only when changed files are limited, schema/runtime/worker risk is absent, focused tests exist, remote product evidence exists, and same-head remote quality-gate succeeds.

It never weakens secret scan, npm failure, placeholder-only evidence, lifeboat-only evidence, runtime readiness, or production readiness boundaries.
