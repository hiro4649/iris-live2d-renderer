# Codex PR Evidence Compact Policy

Version: v0.9.3

PR bodies should carry only the minimum evidence needed to decide merge readiness. Large details move to safe artifacts. PR text must not include raw logs, raw diffs, large generated file lists, secrets, endpoint values, private paths, production data, or personal data.

Required PR body summaries: Goal, Task Contract, Target Scope, Validation summary, Evidence Integrity summary, Human confirmation summary, Residual risks, and safe artifact references when detailed evidence is too large.
