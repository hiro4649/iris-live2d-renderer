# Test Coverage Evidence Policy

Coverage percentage tools are not required. For bug fixes, refactors,
migrations, implementation changes, or behavior changes, PR evidence must map
changed area, risk, test command, what the test covers, edge/failure paths, or a
reason no test was added. Docs-only and policy-only changes are not applicable.
