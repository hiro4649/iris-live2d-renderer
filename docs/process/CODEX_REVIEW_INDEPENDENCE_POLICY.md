# Codex Review Independence Policy

Review evidence must not be only the writer's self-assertion. The harness requires separated writer evidence and reviewer evidence fields, a concrete checklist, negative case coverage, scope boundary confirmation, runtime readiness denial when no oracle exists, and product code changed status aligned with changed files.

The gate checks evidence separation deterministically. It does not claim external human judgment quality or require an external LLM.
