# Codex Bugfix

Use this skill for bugfix work only.

1. Classify the bug and affected area.
2. Reproduce or record why reproduction is not possible before editing.
3. Write the root-cause finding or hypothesis.
4. Fix minimally and avoid unrelated changes.
5. Verify the fix with the smallest meaningful checks.
6. Check nearby similar risk.

Report task mode, reproduction status, root cause, changed files, fix summary,
verification result, remaining risk, and rollback note.
